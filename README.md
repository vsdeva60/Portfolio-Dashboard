# Portfolio-Dashboard
These are the details of my Power BI Project-

First Dashboard- This dashboard [ Madhav Ecommerce Sales]provides actionable insights into key business metrics, including:

2- Category & Payment Analysis – Breakdown by product category and payment mode
3- State-wise Trends – Sales distribution across different regions
4-Time-Based Analysis – Monthly profit trends with drill-down capabilities.
